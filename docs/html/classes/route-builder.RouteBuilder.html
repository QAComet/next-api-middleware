Class RouteBuilder

This class is responsible for wrapping routes with the configured middleware. Typically, RouteMiddleware will construct an instance of this class with the routes method. So

RouteMiddleware
  .register(middleware)
  .routes(routes);

will return an instance of RouteBuilder. It is not recommended to use this class yourself and instead depend on RouteMiddleware instead.
